Understanding File Sizes

File sizes are measured in bytes, where 1 kilobyte (KB) = 1,024 bytes, 1 megabyte (MB) = 1,024 KB, and 1 gigabyte (GB) = 1,024 MB. Understanding file sizes is important for email attachments, storage management, and upload limits.
